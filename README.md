## 資料の構成
本リポジトリには、以下の設計資料が記載されています。

### ①実装機能（要件定義）
- 会員側実装機能
- 管理者側実装機能

### ②機能設計
- 会員側
- 管理者側
  
### ③非機能要件
- 可用性・性能・セキュリティなど、機能以外に求められる要件を整理しています。
  
### ④画面遷移図
- 会員側・管理者側それぞれの画面遷移を図で整理しています。
  
### ⑤ER図
- Draw.io を利用して作成
- テーブル構成およびリレーションを定義
  
### ⑥フローチャート
- 課題として、1機能を対象にフローチャートを作成
